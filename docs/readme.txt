
The key files are:

cla.html        -- the terms of service displayed on the site
cla_faq.html    -- the FAQ for the site
confirmation.md -- this file is sent after the user confirms his or her registration
verification.md -- this file is sent once the user clicks to verify the registration


email_request.md  -- this is the email text to ask someone to accept the CLA
email_reminder.md -- this is the email reminder to ask someone to accept the CLA

issue_request.md -- this is the issue text to ask someone to accept the CLA
issue_reminder.md -- this is the issue text to ask someone to accept the CLA
